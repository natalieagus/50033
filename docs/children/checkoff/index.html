Checkoff

Grading notes

The grading for this lab is binary (completed or not completed) for each feature. This means you should implement each feature fully to obtain the mark for that feature. In this checkoff, the features we are looking for are: meaningful player feedback, multiple scenes in the game, some kind of persistent data (could be highscore, user setting, etc), existence of interactable powerups (can be anything that benefits the player or introduce dilemma), and properly polished game restart functionality.

Don't worry, we are lenient and will give you the marks as long as we see some form of the requested feature.
